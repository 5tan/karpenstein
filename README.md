# Ruiny zamku Karpień - Karl Wehse

Niniejsze repozytorium prezentuje wynik (naiwnego) tłumaczenia książki Karla Wehse nt. zamku Karpień, z 1883 roku.

Tłumaczenia dokonano przy użyciu modelu gpt-5.1. OCR wykonano przy użyciu tesseract (model fraktur).

Link do wersji HTML tłumaczenia: [tutaj](https://5tan.github.io/karpenstein).

Oryginalny tekst na stronach Opolskiej Biblioteki Cyfrowej: [tutaj](https://obc.opole.pl/Content/7596/4762s.pdf).
